# アイエスシー株式会社

**概要**
アイエスシー株式会社は、画像処理システムを活用した製品検査・外観検査機器を開発するIT企業です。1990年2月に設立され、ソフトウェア・ハードウェアの受託開発、画像による各種検査装置開発、映像関連システム開発などを行っています。

**事業内容**
- ソフトウェア・ハードウェアの受託開発
- 画像処理を利用した製品検査・外観検査装置の開発
- 監視カメラなどの画像記録装置開発
- ハイスピードカメラの制御ソフト開発
- Windowsアプリケーション開発
- 映像関連システム開発

**技術スタック**
- プログラミング言語: C++, C, C#, Python
- ハードウェア: カメラ、産業カメラ、ハイスピードカメラ
- 技術: 画像処理、画像認識、GPUによる高速化、AI、機械学習
- OS: Windows, Linux

**主要製品・サービス**
- 色検査システム（塗装やメッキ処理の色ムラや抜け、色調の違いを判定）
- 異物検査システム（工業製品や食品などに異物が混入していないかを確認）
- キズ・汚れ検査システム（製品の細かなキズや汚れなどの不良品検出）
- 形状検査システム（製品の寸法や曲がりの角度、打刻やマークの位置判定）
- 映像解析・物体認識システム
- VR（仮想現実）・AR（拡張現実）技術
