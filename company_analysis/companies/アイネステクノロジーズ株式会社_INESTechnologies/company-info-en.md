# 株式会社アイネステクノロジーズ (INES Technologies Inc.)

## Company Overview

Ines Technologies Co., Ltd. is a pioneering IT outsourcing service company established in 1999. Headquartered in Totsuka-ku, Yokohama City, Kanagawa Prefecture, its main business is contracting data center operations and infrastructure construction, focusing on reducing costs and improving efficiency in customers' IT system operations.

Its strength lies in providing cloud server and client environments utilizing virtualization technology, and it possesses high technical capabilities in building and operating cloud infrastructure using AWS. In particular, it has a proven track record in building and operating server infrastructure for public systems. Its proprietary products include the cloud signage "R-Sign," the virtual thin client solution "FogosPRO," and the cloud server/web system monitoring solution "JENNIFER." The number of employees is approximately 98 (estimated).

Its technical capabilities and track record in providing comprehensive services from design to implementation and operational support in cloud-based device management utilizing Microsoft Intune and Windows Autopilot have been recognized by Microsoft. It has also obtained the Privacy Mark certification.

**Website:** [https://www.iti.ines.co.jp](https://www.iti.ines.co.jp)

## History

- April 1999 — Establishment of I-Nes Technologies Co., Ltd.
- 2000s — Development of data center operation contracting and infrastructure construction businesses
- 2010s — Launch of cloud server/client environment services utilizing virtualization technology
- 2010s — Development of cloud signage "R-Sign", virtual thin client "FogosPRO", and monitoring solution "JENNIFER"
- 2020s — Focus on cloud infrastructure construction and operation utilizing AWS, accumulating experience in infrastructure construction for public systems
- June 2024 — Change of head office address from Nihonbashi Kakigaracho, Chuo-ku, Tokyo to Totsuka-ku, Yokohama City, Kanagawa Prefecture (Registration Information)
- Present — Development of cloud-based device management services utilizing Microsoft Intune/Windows Autopilot

## Services/Products

### Cloud Infrastructure Services

- **Cloud Server Environment Construction** — Provision of cloud server environments utilizing virtualization technology
- **Cloud Client Environment** — Design, construction, and operation of virtual desktop (VDI) environments
- **AWS Infrastructure Construction and Operation** — Design, construction, and operation of cloud infrastructure utilizing AWS
- **Public System Infrastructure** — Construction and operation of server infrastructure for public institutions

### Proprietary Products

- **R-Sign (Cloud Signage)** — Cloud management service for digital signage
- **FogosPRO (Virtual Thin Client Solution)** — Provision of a secure remote desktop environment
- **JENNIFER (Web System Monitoring Solution)** — Monitoring solution for cloud servers and web systems
- **SecureRDP** — Provision of a secure remote access environment

### Device Management Services

- **Microsoft Intune / Windows Autopilot** — Design, implementation, and operation support for cloud-based device management (Microsoft certified)
