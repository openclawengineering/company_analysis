# 株式会社ディーインクリエイティブ

## 会社概要

株式会社ディーインクリエイティブ（DIN CREATIVE）は、神奈川県横浜市を拠点とするITソリューション企業。中小企業の味方として、最先端のAI技術とWEB技術を駆使した問題解決に特化している。クラウドVPS・プライベートクラウド・専用サーバーの提供、コロケーションサービス、SSL証明書販売、メールリレーサービスなど、サーバーインフラからWEBアプリケーション開発まで幅広く手がける。独自のAI駐車場管理システム「AI-Parking」やスマートフォンアプリの開発実績も持つ。

**ウェブサイト:** https://www.d-in.jp/

## 事業内容

### クラウド・サーバー事業
- クラウドVPSサーバー（AlmaLinux等、最短数日で納品）
- プライベートクラウド（完全冗長化、高可用性環境）
- 専用サーバー（1台独占利用）
- 共用サーバー（ライトプラン/スタンダードプラン）
- コロケーションサービス（1U単位でデータセンター設置）
- 独自構成サービス（ロードバランサー、DB冗長化等）

### セキュリティ事業
- 攻撃遮断くん（IPS：不正侵入遮断システム）
- SiteGuard Server Edition（ホスト型WAF）
- SSLサーバー証明書販売（GMOグローバルサイン、ジオトラスト、サイバートラスト、デジサート・ジャパン）

### メール・通信事業
- ベアメール（メールリレーサービス・迷惑メールスコアリング）

### 開発事業
- AI駐車場管理システム「AI-Parking」
- スマートフォンアプリ開発
- HPパッケージ（ホームページ構築パッケージ）
- インターネット・WEBサービス構築

## 特徴

- 中小企業向けITソリューションに特化
