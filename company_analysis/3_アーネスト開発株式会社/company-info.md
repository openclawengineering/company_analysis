# アーネスト開発株式会社

**概要**
アーネスト開発株式会社は、横浜市鶴見区に拠点を置くソフトウェア開発会社です。2019年4月1日に設立され、ソフトウェアエンジニアリング事業を展開しています。

**事業内容**
- C#、JAVA、PHP、Ruby、Perl、COBOLなどのプログラミング言語による受託開発
- WebアプリケーションのWebフロントエンド開発
- 顧客要件の仕様書作成、保守・運用、システムの企画・立案
- その他、システム運用に関するサポートと関連業務

**技術スタック**
- プログラミング言語: C#、C++、Java、PHP、Ruby、Perl、COBOL
- データベース: Oracle、SQL Server
- Webアプリケーション: HTML、JavaScript、CSS
- フレームワーク: SAP、intra-mart
