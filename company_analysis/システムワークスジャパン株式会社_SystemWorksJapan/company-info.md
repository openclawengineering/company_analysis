# システムワークスジャパン株式会社 (System Works Japan Corporation)

## Company Overview

株式会社システムワークス (System Works Japan Corporation) is an IT system development company providing comprehensive software solutions including web applications, embedded systems, and digital signage systems. The company has extensive experience in developing systems for various industries.

**Website:** [https://system-works.co.jp](https://system-works.co.jp)

## History

The company has over 20 years of experience in system development, serving various industries from education to automotive inspection.

## Services/Products

### Web Application Development
- **Tenant Management System**
  - Platform: Windows, AWS (RDS, S3, Step Functions, Lambda), Access
  - Features: Monthly billing, payment tracking, income breakdown management for real estate properties

- **University Portfolio System**
  - Platform: Windows, Linux, MariaDB, Apache, ASP.NET Core, C#
  - Features: Student learning outcome management, class/semester reflection tools, faculty analysis tools

- **University Portal Site**
  - Platform: Linux, PostgreSQL, Apache, PHP, Laravel
  - Features: Federation-based single sign-on, account permission and role management

- **Municipal Groupware System**
  - Platform: Windows, SQLServer, IIS, ASP.NET MVC, C#
  - Features: Schedule management, messaging, file sharing, questionnaire management

### Embedded Systems
- **Digital Signage System (Panoadvance)**
  - Platform: Windows IoT, C#/C++, .NET Framework, Direct3D
  - Features: Flexible layouts, wide format support, emergency earthquake alerts

- **Vehicle Inspection Process Management System**
  - Platform: Windows, PostgreSQL, C#, .NET Framework
  - Features: Complete vehicle inspection workflow management

- **Vehicle Inspection Control Console**
  - Platform: Windows, C#, .NET Framework
  - Features: PLC communication, touch-enabled GUI

- **Scoreboard System**
  - Platform: Windows, MySQL, C#/C++, .NET Framework, Direct3D
  - Features: Team/player management, real-time scoreboard rendering for baseball stadiums
