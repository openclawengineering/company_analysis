# 株式会社アイネステクノロジーズ (INES Technologies Inc.)

## Company Overview

株式会社アイネステクノロジーズは、1999年に設立されたITアウトソーシングサービスの先駆企業である。神奈川県横浜市戸塚区に本社を置き、データセンター運用業務の請負やインフラ構築を主な業務として、顧客のITシステム運用のコスト削減と効率向上に注力している。

仮想化技術を活用したクラウドサーバ環境・クライアント環境の提供サービスを強みとし、AWSを活用したクラウドインフラの構築・運用において高い技術力を持つ。特に公共システムにおけるサーバインフラの構築・運用に実績がある。独自のプロダクトとして、クラウドサイネージ「R-Sign」、仮想シンクライアントソリューション「FogosPRO」、クラウドサーバ・Webシステム監視ソリューション「JENNIFER」などを展開している。従業員数は約98名（推定）。

Microsoft IntuneおよびWindows Autopilotを活用したクラウド型デバイス管理において、設計から導入・運用支援まで一貫したサービス提供が可能な技術力と実績がMicrosoftから評価されている。プライバシーマークも取得済み。

**Website:** [https://www.iti.ines.co.jp](https://www.iti.ines.co.jp)

## History

- 1999年4月 — 株式会社アイネステクノロジーズ設立
- 2000年代 — データセンター運用業務の請負、インフラ構築事業を展開
- 2010年代 — 仮想化技術を活用したクラウドサーバ/クライアント環境サービスを開始
- 2010年代 — クラウドサイネージ「R-Sign」、仮想シンクライアント「FogosPRO」、監視ソリューション「JENNIFER」を展開
- 2020年代 — AWS活用のクラウドインフラ構築・運用に注力、公共システムのインフラ構築に実績を蓄積
- 2024年6月 — 本店住所を東京都中央区日本橋蛎殻町から神奈川県横浜市戸塚区に変更（登記情報）
- 現在 — Microsoft Intune/Windows Autopilot活用のクラウド型デバイス管理サービスを展開

## Services/Products

### クラウドインフラサービス
- **クラウドサーバ環境構築** — 仮想化技術を活用したクラウドサーバ環境の提供
- **クラウドクライアント環境** — 仮想デスクトップ（VDI）環境の設計・構築・運用
- **AWSインフラ構築・運用** — AWSを活用したクラウドインフラの設計・構築・運用
- **公共システムインフラ** — 公共機関向けサーバインフラの構築・運用

### 独自プロダクト
- **R-Sign（クラウドサイネージ）** — デジタルサイネージのクラウド管理サービス
- **FogosPRO（仮想シンクライアントソリューション）** — セキュアなリモートデスクトップ環境の提供
- **JENNIFER（Webシステム監視ソリューション）** — クラウドサーバ・Webシステムの監視ソリューション
- **SecureRDP** — セキュアなリモートアクセス環境の提供

### デバイス管理サービス
- **Microsoft Intune / Windows Autopilot** — クラウド型デバイス管理の設計・導入・運用支援（Microsoft認定）
