## Business Lines

### Solutions

- **勤時 (Kintoki)** — Attendance and work hours management SaaS. Supports labor management efficiency and employee health. Integrated with facial recognition ID "FreeiD".
- **CLOUDIAN HyperStore** — Object storage solution for sustainably managing ever-growing data volumes
- **Neuron Enterprise Search** — Turns scattered information into assets; enterprise search solution with generative AI integration
- **Knowledge Stack** — Secure AI chat service built on Microsoft Azure (private ChatGPT for organizations)
- **RADIUS Solution for Telecom Carriers** — High-speed, high-availability, scalable RADIUS servers
- **System Reform** — Legacy system modernization and migration services
- **Generative AI** — GitHub Copilot integration, voice AI applications
- **Unicage Development Method** — Ultra-high-speed development methodology (originating in Japan)
- **IoT Device Monitoring** — Real-time state monitoring, video distribution, system state monitoring
- **Single Sign-On (SSO)** — Cloud-based ID management and integrated authentication (Okta CIC)
- **Security Solutions** — Ransomware countermeasures, secure migration
- **Education Sector Systems** — Specialized solutions for universities and research institutions

### Contract Development
Application development centered on public, social, financial, and industrial sectors.
