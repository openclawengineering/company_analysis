# 株式会社八芝ソフトエンジニアリング / Yashiba Software Engineering

## Overview

Yashiba Software Engineering is a Yokohama-based software development company, part of the Yashiba Group (alongside Yashiba Transport). The company focuses on software development, system design and maintenance, and IT staffing dispatch services. It actively recruits developers regardless of prior experience.
