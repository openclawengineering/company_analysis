# 株式会社ライトウェア / Lightware

## Overview

Lightware is a Tokyo-based software development company specializing in web application development, particularly using Java and ColdFusion. The company offers system development, design, and staffing services, and operates a matching site called "Light Staff" (スタートライトスタッフ) for IT project opportunities.
