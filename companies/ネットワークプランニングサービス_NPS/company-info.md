# ネットワークプランニングサービス株式会社 / Network Planning Service Inc. (NPS)

## Overview
Network Planning Service (NPS) is a Kanagawa-based IT company offering web design, online shop construction, system development, server construction, monitoring camera systems, and PC support. They also developed the "Mimamoro" (見守ろう) elderly monitoring system used in nursing care and disaster recovery applications, including the "Kamaishi Recovery Live Camera" project.

## Key Details
- **Website:** https://www.nps-inc.jp/

## Business Areas
- **Website Creation:** Homepage design and development
- **Online Shop Construction:** E-commerce site building
- **System Development:** Custom software development
- **Server Construction:** Server setup and infrastructure
- **Monitoring Camera Systems:** Security and monitoring solutions
- **PC Support:** Technical support services
- **Design:** Graphic and web design services

## Notable
- Developed "環境戦士 見守朗" (Environment Warrior Mimamoro) — elderly monitoring/safety check system
- Listed by Kanagawa Prefecture as a welfare monitoring service provider
- Kamaishi Recovery Live Camera project for disaster recovery (donated to Kamaishi City mayor)
- Multiple self-operated websites: mimamoro.com, projectkobo.com, k-music-s.com, nursery-shonan.com
- 3D panorama video production capability
- Website appears not updated since 2016
