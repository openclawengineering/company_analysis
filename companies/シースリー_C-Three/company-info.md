# 株式会社シースリー / C Three Inc.

## Overview
C Three Inc. is an IT company specializing in server/database infrastructure design and construction for large-scale user systems, enterprise information system operation and maintenance, and web application development. Founded on three core principles: Connection (creating opportunities), Communication (maintaining relationships), and Collaboration (business co-existence and co-prosperity).

## Business Areas
- **Server Infrastructure Solutions:** OS and middleware infrastructure solutions for large-scale user systems, including vendor-based and open-source servers
- **Enterprise IT Services:** System operation and maintenance for enterprise server infrastructure, including version upgrades and incident response
- **Web Application Development:** Corporate website development and related technical projects
- **Database Architecture:** Database design, construction, migration, and maintenance (Oracle, vendor-based, and open-source databases)
