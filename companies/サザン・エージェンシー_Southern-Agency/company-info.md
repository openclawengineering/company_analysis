# サザン・エージェンシー / Southern Agency

## Overview

Kawasaki-based software and hardware development company founded in 1990, specializing in embedded systems, business applications, and web development. Accepts small-scale and one-off projects that larger firms may decline.

## Key Details

- **Website:** https://www.sacgrp.co.jp/
- **Address:** 210-0014 Kawasaki, Kawasaki-ku, Kaizuka 1-4-16, LM Kawasaki Kaizuka 2F
- **Founded:** 1990
- **Business Areas:** Custom software development, embedded software, IT processing services, network/communications, SES

## Services

- Embedded system software development
- Hardware design development
- Business system development (attendance, sales, inventory, production management, etc.)
- Web application development
- Custom software from scratch to customizations
- Excel/VBA-based report creation
- General-purpose board control programs and related applications
