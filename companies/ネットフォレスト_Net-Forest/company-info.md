# 株式会社ネットフォレスト / Net Forest Inc.

## Overview
Net Forest is a Yokohama-based comprehensive IT company operating from "communications to design." As a long-established internet service provider, they offer infrastructure services (networking, cloud, servers), web design/development, and specialized solutions including hotel information systems and security products. They maintain a dedicated support center in Asahikawa, Hokkaido.

## Key Details
- **Address:** Yokohama (exact address on corporate page)
- **Website:** https://www.netforest.ad.jp/
- **Support Center:** Asahikawa, Hokkaido

## Business Areas
- **Communications/Network:** FLET'S line connectivity, dedicated lines/VPN, router configuration, Wi-Fi deployment, network security, domain/DNS management, data center services
- **Cloud Services:** Cloud server construction, cloud connectivity, cloud platform security
- **Web Application Servers:** Web server construction, capacity design, web infrastructure security, application runtime environments
- **Web Design & Consulting:** Website creation, CMS construction, operation/update outsourcing, marketing/promotion, website security
- **Hotel Information Systems:** In-room information/VOD systems for hotels
- **Internet Service Provider:** ISP services (long-established)
- **Hosting Services:** Web hosting
- **Security Solutions:** Website security and network security products
